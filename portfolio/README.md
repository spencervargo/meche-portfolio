# Spencer Vargo — Mechanical Engineering Portfolio

A simple, fast, responsive portfolio to showcase projects, skills, and certifications.

## Getting started

Open `index.html` in a browser (double-click or use a local server).

- Main file: `index.html`
- Styles: `styles.css`
- Scripts: `main.js`
- Images: `assets/img/`

## Customize

- Name and tagline: Edit the hero section in `index.html`.
- Projects: Duplicate a project `article.card` inside `#projects .projects-grid`, update `data-tags` (`design`, `analysis`, `manufacturing`) and content.
- Skills: Update the chips under each skill group in `#skills`.
- Certifications: Add new `<li>` blocks in the `#certifications .timeline`.
- Contact links: Update email, LinkedIn, resume path in the `#contact` section.

## Theme

Click the moon icon to toggle light/dark. The setting persists in `localStorage`.

## Deploy

- GitHub Pages: push the `portfolio` folder to a repo and enable Pages (root).
- Netlify/Vercel: deploy as a static site from the `portfolio` directory.
